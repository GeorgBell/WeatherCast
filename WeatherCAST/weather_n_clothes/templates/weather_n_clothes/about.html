About the project

WeatherCast app developed in the course of entrance task for CTO school

Author: George Kolokolnikov

Project name: Smart weather forecast app

Level: Asterisk task

Service development:

  • Tech stack: Python programming language and Django framework
  • UI: Website
  • Response format: Response from weather API is represented on the main page. Weather pic, temperature and description is shown

Operation pipeline:

  1. User enters the website
  2. User location is determined by IP
  3. The detected city is added to string that is used as request to weather API
  4. Response from API is parsed
  5. Weather description is used to define the appropriate outfit
  6. The data is rendered on the main page
  7. If user wants to change the city, he/she presses the button
